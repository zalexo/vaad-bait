# Финансы

## Реестры

- `payments.xlsx` — начисления, оплаты, задолженности и переплаты жильцов.
- `expenses.xlsx` — расходы домового комитета.

Одна строка должна описывать одну операцию. Не удаляйте старые записи:
корректировки добавляются отдельно с пояснением.

Статусы:

- `подтверждено` — есть документ или прямое подтверждение пользователя;
- `требует проверки` — сведения ещё нужно сверить;
- `отменено` — запись сохранена для истории, но не участвует в итогах.

Подтверждающие документы хранятся в `documents/invoices/` и
`documents/receipts/`. В реестре указывается относительный путь к файлу.

Пустые шаблоны можно восстановить командой:

```sh
python3 scripts/create_finance_templates.py
```

По умолчанию команда не перезаписывает существующие реестры. Параметр
`--force` можно использовать только после отдельной резервной копии и
подтверждения.
