# Документы

В этой папке хранятся документы домового комитета:

- счета и квитанции;
- договоры;
- коммерческие предложения;
- письма и уведомления;
- фотографии и технические материалы.

Рекомендуемый формат имени:

`YYYY-MM-DD-краткое-название.ext`

Не добавляйте временные файлы, дубликаты и документы, не относящиеся к
работе домового комитета.
