# Собственники квартир

Актуальный список собственников квартир и их телефонных номеров хранится в
[owners.xlsx](owners.xlsx).

## Правила ведения

- Телефон записывается в локальном израильском формате.
- Для каждой квартиры используется отдельная строка.
- Номер квартиры — главный ключ для связи со списком фактических жителей
  в [residents.xlsx](residents.xlsx).
- Род занятий указывается в колонке `עיסוק` только если он известен
  от пользователя или из надёжного документа.
- Если по квартире есть ещё один собственник или контактное лицо,
  указывать его в колонках `איש/אשת קשר נוסף/ת` и `טלפון נוסף`,
  не удаляя основную запись без отдельного подтверждения.
- После изменения данных обновляется дата в верхней части таблицы.
- Доступ к файлу предоставляется только членам домового комитета.
- Старые копии списка с персональными данными не добавляются в репозиторий.
