# Домовой комитет

Рабочий репозиторий домового комитета: жильцы, протоколы собраний,
принятые решения, платежи, расходы, подрядчики и общие документы.

## Структура

- `residents.xlsx` — актуальный список жильцов и телефонов.
- `residents.md` — описание правил ведения списка жильцов.
- `finance/payments.xlsx` — начисления и оплаты жильцов.
- `finance/expenses.xlsx` — расходы домового комитета.
- `contractors/` — садовник и другие подрядчики.
- `taxes/YYYY/` — налоговые материалы и чек-листы по годам.
- `meetings/` — протоколы собраний и их шаблон.
- `decisions/decisions.md` — журнал принятых решений.
- `documents/invoices/` — счета поставщиков.
- `documents/receipts/` — квитанции и подтверждения оплаты.
- `documents/` — договоры, предложения и другие документы.

## Основные правила

1. Не публиковать содержимое репозитория вне домового комитета.
2. После изменения списка жильцов обновлять дату в `residents.xlsx`.
3. Для каждого собрания создавать отдельный файл в `meetings/`.
4. Итоговые решения переносить из протокола в журнал решений.
5. Давать файлам даты в формате `YYYY-MM-DD`.
6. Не отмечать платёж или расход подтверждённым без источника.
7. Перед отправкой сообщений и любыми внешними действиями получать
   подтверждение пользователя.

## Именование файлов

- Протокол: `meetings/YYYY-MM-DD.md`
- Документ: `documents/YYYY-MM-DD-краткое-название.ext`
