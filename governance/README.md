# Управление домовым комитетом

Здесь хранятся рабочие документы по выбору и работе домового комитета
(ועד בית).

## Файлы

- [owner-signatures-2026.xlsx](owner-signatures-2026.xlsx) — отметки
  подписей собственников за выбор нового ועד.

## Правила ведения подписей

- Одна строка соответствует одной квартире и одному собственнику из
  [owners.xlsx](../owners.xlsx).
- В колонке `קישור לבעלים` указывать связь с реальным владельцем:
  `owners.xlsx / דירה N`.
- Если в `owners.xlsx` есть дополнительный собственник или контактное
  лицо, переносить его в таблицу подписей для удобства обзвона.
- По умолчанию статус подписи — `לא נחתם`.
- После получения подписи указывать статус, дату, способ получения и
  ссылку на документ или фото, если оно сохранено в репозитории.
- Не отмечать подпись полученной без явного подтверждения пользователя
  или сохранённого документа.
- Номер квартиры — главный ключ связи с собственником.
